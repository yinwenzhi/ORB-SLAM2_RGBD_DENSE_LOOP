    message_filters::Subscriber<sensor_msgs::Image> depth_sub(nh, "/camera/depth/image_raw", 1);
上面的/camera/rgb/image_raw修改成摄像机对应的色彩图片话题
/camera/depth/image_raw修改成深度图片
因为不同设备话题名称不一样
